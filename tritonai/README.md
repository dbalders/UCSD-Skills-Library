# TritonAI Skills

This folder is for skills maintained by the TritonAI or UCSD AI Tools team.

Skills here should be safe to publish in the public UCSD Skills Library.

Each skill should live in its own folder:

```text
tritonai/
  example-skill/
    SKILL.md
    references/
    assets/
    scripts/
```

Only `SKILL.md` is required. Keep internal-only workflows, deployment procedures, private infrastructure details, and restricted data handling in `UCSD-Skills-Library-Secure` instead.
